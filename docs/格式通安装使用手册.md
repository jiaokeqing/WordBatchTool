# 格式通安装使用手册

## 1. 软件简介

格式通是一款离线批量 Word 排版工具，支持选择 `.doc` / `.docx` 文件或文件夹，套用本地格式模板，批量输出 DOCX、PDF 和 ZIP 结果包。

适用场景：

- Windows 办公电脑：优先使用 WPS / Microsoft Office 转换引擎。
- 华为 Windows 电脑：使用 Windows x64 安装包或绿色版。
- 华为统信 UOS / ARM64：需在目标设备本机构建 Linux ARM64 版本，并优先使用 LibreOffice headless。

## 2. 安装包与绿色版

Windows 安装包路径：

```text
release/windows-x64/格式通_1.0.0_x64-setup.exe
```

Windows 绿色版路径：

```text
release/windows-x64/格式通.exe
release/windows-x64/word-batch-sidecar.exe
```

华为 Windows x64 绿色版路径：

```text
release/huawei-windows-x64/格式通_1.0.0_x64-setup.exe
release/huawei-windows-x64/格式通.exe
release/huawei-windows-x64/word-batch-sidecar.exe
```

绿色版必须保持两个 exe 在同一目录内，不要只复制主程序。

## 3. Windows 安装与启动

1. 双击 `格式通_1.0.0_x64-setup.exe`。
2. 按安装向导完成安装。
3. 从桌面或开始菜单启动“格式通”。
4. 正常启动后只显示格式通主窗口，不应再额外显示终端窗口。

运行要求：

- Windows 10/11。
- 已安装 WebView2 Runtime。
- 如需 `.doc` 转 `.docx` 或 PDF 导出，建议安装 WPS Office 或 Microsoft Office。

## 4. 绿色版启动

1. 打开 `release/windows-x64/` 或 `release/huawei-windows-x64/`。
2. 双击 `格式通.exe`。
3. 确认同目录存在 `word-batch-sidecar.exe`。

如果主界面提示“本地服务未就绪”，请确认 sidecar 文件未被杀毒软件隔离，并重新启动软件。

## 5. 基本使用流程

1. 进入“处理任务”。
2. 点击“选择 Word 文件”或“选择文件夹”。
3. 在“选择模板”中选择默认模板或自定义模板。
4. 根据需要开启或关闭“导出 PDF”。
5. 点击“开始处理”。
6. 在“任务记录”中查看进度。
7. 任务完成后点击“下载 ZIP”，选择保存位置。

## 6. 模板管理

在“模板库”中可以：

- 新建模板。
- 复制内置模板后编辑。
- 设置默认模板。
- 删除非内置、非默认模板。

内置模板不能直接修改，需复制后编辑。

## 7. 平台适配

在“平台适配”中可以查看当前系统和转换引擎状态：

- Windows：优先检测 WPS / Microsoft Office。
- 统信 UOS / Linux：优先检测 LibreOffice。
- 未安装转换引擎时，部分 PDF 导出或 `.doc` 转换任务会失败，但 DOCX 排版仍可使用。

## 8. 常见问题

### 启动后出现终端窗口

新版 sidecar 已使用无控制台模式打包。若仍出现终端窗口，请确认使用的是本次重新生成的安装包或绿色版。

### 页面缩放后任务记录列表显示异常

新版界面已对任务列表和文件结果列表增加列宽保护与横向滚动。在高缩放比例下，列表会优先滚动，不会挤压到不可读状态。

### 下载 ZIP 没有浏览器下载栏

桌面版使用原生保存对话框。点击“下载 ZIP”后请选择保存路径。

### PDF 导出失败

请先确认已安装 WPS Office 或 Microsoft Office，并可正常打开 Word 文档。统信 UOS / Linux 请安装 LibreOffice。

## 9. 构建命令

Windows / 华为 Windows：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\package_desktop.ps1 -Target windows
```

华为统信 UOS / ARM64：

```bash
scripts/package_desktop.sh huawei-uos-arm64
```

UOS / ARM64 版本必须在目标设备本机生成。
